# Country data attribution

`countries.json` in this folder is a trimmed derivative (common/official/Arabic
name, capital, region, subregion, ISO 3166-1 alpha-2 code) of the **mledoze/countries**
dataset:

- Source: https://github.com/mledoze/countries
- License: **Open Database License (ODbL) v1.0** — https://opendatacommons.org/licenses/odbl/1-0/

The data is used here to demonstrate client-side fuzzy search. Per the ODbL,
this derivative is made available under the same license, with attribution to
the upstream project.

Flag images are loaded at runtime from **flagcdn.com** (https://flagcdn.com),
keyed by the alpha-2 code; they are not redistributed in this repository.
